📡 API REST - ENDPOINTS PRINCIPALES
═════════════════════════════════════════════════════════════════════════════

  PROYECTOS:
  POST    /proyectos/crear              Crear nuevo proyecto
  GET     /proyectos                    Listar todos
  GET     /proyectos/{id}               Obtener completo
  PUT     /proyectos/{id}               Actualizar
  DELETE  /proyectos/{id}               Eliminar

  CONTENIDO:
  PUT     /proyectos/{id}/transcripcion Actualizar transcripción
  PUT     /proyectos/{id}/resumen       Actualizar resumen
  PUT     /proyectos/{id}/preguntas     Actualizar preguntas

  EXÁMENES:
  POST    /proyectos/{id}/examenes      Crear examen
  GET     /proyectos/{id}/examenes      Listar exámenes
  GET     /alumnos/{nombre}/examenes    Exámenes de alumno
  GET     /alumnos/{nombre}/estadisticas Desempeño

  BÚSQUEDA:
  GET     /buscar/titulo?q=...          Por título
  GET     /buscar/rol?rol=docente       Por rol
  GET     /buscar/etiqueta/{etiqueta}   Por etiqueta
  GET     /buscar/dias?dias=7           Últimos N días
  POST    /buscar/avanzada              Búsqueda combinada

  ESTADÍSTICAS:
  GET     /estadisticas/generales       Sistema
  GET     /estadisticas/proyecto/{id}   Proyecto
  GET     /estadisticas/etiquetas       Trending
  GET     /estadisticas/alumnos         Alumnos activos

  IMPORT/EXPORT/BACKUP:
  GET     /exportar/proyecto/{id}       ZIP del proyecto
  POST    /importar                     Desde ZIP
  POST    /backup                       Crear backup
  GET     /backups                      Listar backups


🗄️ BASE DE DATOS
═════════════════════════════════════════════════════════════════════════════

  TABLAS SQLite:
  
  proyectos
  ├─ id (UUID)
  ├─ titulo (string)
  ├─ rol (docente/alumno)
  ├─ perfil_edad (niño/adolescente/adulto)
  ├─ fecha_creacion (ISO)
  ├─ duracion_segundos (int)
  ├─ ruta_carpeta (path)
  └─ Índices: fecha, rol, titulo
  
  examenes
  ├─ id (UUID)
  ├─ proyecto_id (FK)
  ├─ alumno_nombre (string)
  ├─ fecha (ISO)
  ├─ nota (0-10)
  ├─ preguntas_acertadas (int)
  ├─ preguntas_totales (int)
  ├─ preguntas_falladas_json (JSON)
  └─ Índices: proyecto, nota, alumno
  
  etiquetas (categorización)
  ├─ id
  └─ nombre (UNIQUE)
  
  proyectos_etiquetas (muchos a muchos)
  └─ Relación flexible
  
  configuracion (global)
  └─ clave-valor

  ESTRUCTURA JSON:
  
  user_data/projects/{uuid}/
  ├─ metadata.json              (2 KB)
  ├─ transcripcion_completa.json (500 KB)
  ├─ resumen.json               (3 KB)
  ├─ preguntas.json             (2 KB)
  ├─ examen_historial.json      (5 KB)
  └─ audio/
     ├─ audio_original.wav      (100+ MB)
     └─ audio_procesado.wav     (50 MB)

🎯 CASOS DE USO
═════════════════════════════════════════════════════════════════════════════

  DOCENTE:
  1. Graba clase (30 min audio)
  2. Carga a sistema
  3. Se procesa automáticamente
  4. Genera transcripción + resumen + preguntas
  5. Exporta como ZIP
  6. Comparte con alumnos
  7. Ve estadísticas de exámenes

  ALUMNO:
  1. Importa ZIP de clase
  2. Lee contenido
  3. Realiza examen (5-10 preguntas)
  4. Obtiene nota inmediata
  5. Ve explicaciones de fallos
  6. Sistema registra desempeño
  7. Puede intentar de nuevo

  ADMINISTRADOR:
  1. Revisa estadísticas globales
  2. Busca "clases de matemáticas este mes"
  3. Identifica alumnos con bajo rendimiento
  4. Prepara refuerzo
  5. Crea backups de seguridad
  6. Exporta reportes
